How to easily install any software in Windows with Chocolatey

Posted on Wed 15 July 2020 in Blog

Package Managers

As per wikipedia “A package manager or package-management system is a collection of software tools that automates the process of installing, upgrading, configuring, and removing computer programs for a computer’s operating system in a consistent manner”. In a nutshell a package manager makes your life easier when you have to install, update or uninstall software programs. Linux users use apt-get as a package manager while Homebrew is the go to option for Mac users. Although Microsoft has provided a Windows Package Manager but that is in beta and is still under development. Therefore, chocolatey is our best option for Windows.

Why do I need a package manager

Let us look at a scenario where you buy a new computer and have to quickly install all your software. Under normal circumstances, you would go to the software’s website and download the set up and then follow certain guidelines to install your desired program. Then you would repeat this step for all your programs. In case something goes wrong and you have to remove the software, you would then need to manually remove all installed files. With a package manager like chocolatey you can do all this in a few seconds.

Installing Chocolatey

All right so now for the interesting part; Installing chocolatey is not complicated at all. Open windows powershell as an administrator. Once there, just copy and paste the following line of code into your powershell:

Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))

This should install chocolatey on your computer. If you run into any complications, just visit https://chocolatey.org/install and follow the instructions there. If everything goes smoothly close your Powershell and you are all set to start using chocolatey.

Installing Single Software

Installing software is as easy as a one liner code. Open command prompt or powershell as an administrator and type the following and press enter:

choco install package_name -y

Of course replace package_name with the name of the actual package you want installed. You can find the list of available packages over at https://chocolatey.org/packages. For example if you wanted to install google chrome, you would use the following command:

choco install googlechrome -y

The -y option at the end stands for yes and it means you will not be prompted to accept license agreements etc and it automatically accepts them for you. There is a way to make this setting the default where you won’t have to type -y everytime and would make the behavior default. However, I would recommend sticking to typing -y at the end of your commands for now. If you do not enter the -y flag at the end of the script, you will be prompted for accepting the agreement which you can then do by typing y and pressing enter.

Installing Multiple Software at once

Installing multiple software is as easy as adding the names one after the other separated by a space after choco install like:

choco install package1_name package2_name package3_name -y

For example the following line will install firefox, google chrome, skype, vlc media player and gimp:

choco install firefox googlechrome skype vlc gimp -y

All these programs will be automatically downloaded and installed.

Updating Packages

Updating packages is very easy. If you want to update a certain package for example vlc; you type the following:

choco upgrade vlc -y

And to update all the installed packages:

choco upgrade all -y

Uninstalling Packages

Uninstalling a package is as easy as

choco uninstall package_name -y
