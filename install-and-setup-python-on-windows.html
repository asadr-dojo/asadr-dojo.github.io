Getting Started with Python on WIndows

Posted on Thu 10 September 2020 in Blog

Introduction

Hey Everybody!

In this tutorial we are going to go over how you can get started with python quickly and easily. I’ll be using Windows in this tutorial and you can follow along easily if you are on windows. However, if you are using Linux or Mac, you can follow along after the installation step. I will give you similar ways of installation for those operating systems. Without further ado let’s get started

Install Chocolatey

The first step in using python is installation. I will be using the anaconda distribution of python as it is one of the most famous ones out there and comes with quite a few useful packages pre-installed. Also, I will use chocolatey to install python. If you want to install chocolatey on your machine, follow the instructions on the chocolatey’s website. Here is a quick overview of how to do this.

Step 1: Open Powershell as Administrator by searching in the start menu

Step 2: Copy and paste the following line of code and press enter:

Set-ExecutionPolicy Bypass -Scope Process -Force; [System.Net.ServicePointManager]::SecurityProtocol = [System.Net.ServicePointManager]::SecurityProtocol -bor 3072; iex ((New-Object System.Net.WebClient).DownloadString('https://chocolatey.org/install.ps1'))

This should install chocolatey on your system.

Installing Python

The next step is to install the anaconda distribution of python. We will do this with chocolatey. All you need to do is open powershell as administrator and execute the following command.

choco install anaconda3 --params '"/AddToPath"'

This should install python 3 via the anaconda distribution. In order to verify installation, type python in powershell:

python

If you don’t see any error messages; you are all set to start using python. Make sure to close and reopen powershell in case you get any error messages. As an alternative to closing and reopen shell, you can also run refreshenv command in powershell:

refreshenv
